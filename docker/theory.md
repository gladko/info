 Docker is a complete container platform that includes:
 - A container runtime (originally using runc under the hood).
 - Tools for building container images (docker build).
 - Image distribution (Docker Hub).
 - A user-friendly CLI and API.
 - Orchestration features (Docker Swarm).
- Docker provides an entire ecosystem for container creation, management, and distribution.
- Docker Engine included both the runtime and image management, making it a one-stop solution for container workloads.
